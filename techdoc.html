Introduction to HTML5, CSS, and JavaScript
  • HTML is the standard markup language used to create the structure and content of web pages.
  • HTML5 is the latest version of HTML, and it introduces new features and improvements over its predecessors.
  • It provides tags and attributes that define the various elements on a web page, such as headings, paragraphs, images, links, forms, and more.

CSS (Cascading Style Sheets):

  • CSS is used to style and layout the visual presentation of HTML Documents.
  • It allows you to control the colors, fonts, spacing, and positioning of elements on a web page.
  • CSS is applied either internally within an HTML document, externally in a separate stylesheet file, or inline within HTML tags.

JavaScript

  • JavaScript is a scripting langauge that enables dynamic and interactive behavior on web pages.
  • It can manipulate the content and structure of HTML, handle user input, and communicate with servers (Ajax).
  • JavaScript is commonly used to create features like sliders, form validation, pop-up messages, and more.
  • It can be included directly within HTML documents or linked externally.

    • These three technologies (HTML5, CSS, JavaScript) work together to create modern, dynamic, and visually appealing websites and
    web applications. As you progress in web development, you'll likely use frameworks and libraries that build upon these
    core technologies, such as React, Angular, or Vue for JavaScript, and Bootstrap or Tailwind CSS for styling.
10 Advanced Git Commands
    1. Add & Commit
  • You have added and committed the files this way multiple times, but what if I tell you it can be done in one line with the `-am` flag?

    $ git add . $ git commit -m "new project"

    • 2. Ammend
  • You can rename your current commit message using the `--amend` flag and write the new message. This will help you with accidental messages.

    $ git commit --amend -m "Love"

    • 3. Revert
  • To undo a commit in Git, you can use the `revert` command. However, this command does not remove any commits. Instead, it creates a new commit that
    undoes the changes made by the original commit. We will use the `log` with `--oneline` flag to view commit in a shorter form.

    f425059 (HEAD -> master, origin/master) Love 8f184d5 first commit
Windows Subsystem for Linux S

List of Commands

  • wsl --install
    Installs Linux on your computer.
  • wsl
    Opens Linux in your terminal.
  • logout or exit
    Returns to command prompt in windows.
  • clear
    Clears the terminal screen.

    • Apache Commands

  • sudo apt update
    Updates Linux before installation of Apache.
  • sudo apt upgrade
    Upgrades Linux.
  • sudo apt-get install apache2
    Installs Apache server.
  • sudo service apache2 start
    Starts Apache server.
  • sudo service apache2 stop
    Stops Apache server.

    • Apache Setup

  • sudo nano
    Open text editor.
  • sudo chown -R username path/to/working/directory
    Make folder writable.
  • sudo nano etc/apache2/sites-available/000-default.conf
    Enable site look for document root and change it your desired folder to run in locations.
  • sudo systemct1 status apache2
    Check if Apache is running.
  • sudo systemct1 restart apache2
    Restart Apache. You need this every time you change the document root. The refresh your browser to update the output.

    • Folder Locations

  • etc/apache2

  • /var/www
